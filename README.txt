-DESCRIPTION-
Netflix Clone application

This following project is a clone of the popular Netflix website.
The application allows users to add and remove movies from their watchlist.

Full Movies CAN NOT be streamed on this application. 

-INSTALLATION-
In order to use the application the user will have to download the project from GitHub and then run the program using Visual Studio Code.
All data that the program uses will be saved to their local storage.
